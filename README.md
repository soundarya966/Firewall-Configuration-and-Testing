# Firewall Configuration & Testing

### Objective
To configure and test a firewall using UFW on Linux, allowing and blocking traffic, monitoring logs, and documenting firewall rules.

### Tools Used
- UFW (Uncomplicated Firewall)
- Linux (Kali / Ubuntu)

### Steps Performed
1. Enabled UFW and set default rules
2. Allowed SSH, HTTP, and HTTPS ports
3. Blocked unused ports
4. Blocked a malicious IP address
5. Enabled firewall logging
6. Tested allowed and blocked connectivity
7. Verified firewall logs

### Outcome
This task demonstrates hands-on firewall management skills, traffic filtering, and security rule documentation.
